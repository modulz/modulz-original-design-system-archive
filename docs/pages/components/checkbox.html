Checkbox

Checkboxes allow the user to select multiple options from a set.

Markup structure

It's not currently possible to directly style a checkbox in modern browsers, so we use this simple hack. We use a <input type="checkbox"> element to provide the necessary functionality but hide it with CSS. Then we use a <div> to provide the styling. Finally, we wrap everything in a <label> to allow interaction.

Checkbox with label

You can add a Label component to toggle the Checkbox component by specifying the relevant checkbox id in the Label's for attribute. Then build a layout by wrapping everything in a Flexbox component—applying spacing with an mr-1 utility class.

By handling the layout independently, you're free to build more complex layouts around your checkbox.

Disabled state

If a checkbox is currently disallowed for any reason, you can apply a disabled state using the disabled attribute.

You can also disable an accompanying Label, using the is-disabled class.

Checkbox group

You can group checkboxes together by wrapping them in an additional Flexbox component—again, using margin utility classes to provide spacing. Make sure to apply the same name attribute value to each checkbox in the same group.

Alternatively, you can stack checkboxes by addding a fd-column utility class to the outer Flexbox component and adjusting the margin utility classes accordingly.
