Heading

A heading summarises a section. There are six possible tags—each representing a semantic level.

Heading tags

A Heading can use any HTML tag from h1 through h6.

I am a default heading

Sizes

There are 9 sizes available. Any size can be used with any HTML tag. The default size is size3.

Responsive sizes

You can toggle sizes as the viewport width changes, using the four breakpoint class prefixes bp1-, bp2-, bp3- and bp4-. In the example below, the Heading scales from size7 to size8, then finally to size9. Resize your browser window to see the changes take effect.

Leading

By default, a Heading has no line spacing. Leading can be applied using the line-height utility classes. If a Heading uses responsive sizes, make sure to apply an appropriate line-height utility class for each relevant breakpoint. Pro tip: when setting a Heading, test it with a long string of placeholder text to check the leading at each breakpoint.

I am a really really unreasonably long heading so you can test my leading.

I am a really really unreasonably long heading so you can test my leading.

Weight

By default, headings are bold. For a lighter weight, add a thin class.
