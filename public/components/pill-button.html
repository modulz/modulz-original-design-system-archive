Pill Button

Pill Buttons allow users to take significant action–submit a form, follow a user, create a record etc.

Pill button

Occasionally, if an action is more significant, essential or impactful, one of three pronounced buttons can be used.

Colors

You can specify one of 12 color variations. A coloured PillButton does not hold any semantic value and should be chosen to support your brand.

There are also grayscale and metallic color variations.

There are also social media color variations.

Sizes

There are two sizes available by adding a size1 or size2 class. Size 1 is the default.

Button with data

You can add an icon to support the PillButton text and provide visual reinforcement. For best results, we recommend Feather icons. The icon is wrapped in a div to provide the necessary margin.

Or you can put an icon on the right.

If an icon is sufficient on its own, you can remove the button text.

Active state

An active state, which can be applied by adding an is-active class. Example use cases might include when a PillButton is currently performing an action or waiting for data to load.

Disabled state

If an action is usually possible but is currently disallowed for some reason, you can apply a disabled state using the is-disabled class. As shown below, different markup is required depending on the html tag.

Tags

A variety of html tags can be used to implement a PillButton, each with their own unique purpose.
