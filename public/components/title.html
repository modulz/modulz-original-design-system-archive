Title

A small heading set in all caps for emphasis. A Title summarises a section and can be used either on its own or in conjunction with another Heading.

Default title

Titles do not wrap unto a new line, so use only very short strings of text.

Responsive sizes

Each size can be toggled as the viewport width changes, using the 4 responsive prefixes. Resize your browser window the see the title adapt.

Colors

Colors are handled with utility classes. The recommended color values are 600 for light backgrounds and 300 for dark backgrounds. Check out the color docs for more detailed guidelines on color usage.
